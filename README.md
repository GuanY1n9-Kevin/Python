## 🏭 1. 工业纺丝工艺计算器

这是我的第一个作品，专门为了解决纺丝、化纤、熔融纺丝车间里复杂的公式死算问题。软件基于 Python 的 `CustomTkinter` 框架开发，界面支持深色模式，排版完全贴合一线工艺工程师的调机习惯。

### 🌟 核心功能亮点
* **智能联动换算**：多个模块支持“输入1个已知项，留空其余项”的自动补全，不需要死记公式。
* **随输随显（实时计算）**：8号功能区采用了实时监听，数字一变，结果立刻刷新，不用频繁点按钮。
* **硬件规格防呆**：直接把工厂 1~15 号机台的计量泵规格写进了代码，下拉菜单直接调用。

### ⚙️ 包含的 9 大核心功能：
* **1. 纤度计算 (圈数)**：摇黑板/缕纱机称重后快速换算 D 和 dtex。
* **2. 纺丝产能预估**：输入卷绕速度、丹尼、锭数，一键预估小时产能和设定时间总产量。
* **3. 长重双向转换**：已知长度求千克/克，或者已知克重逆推长度，智能填一补二。
* **4. 变频器比例换算**：改纺微调规格时，根据老规格和老频率快速逆推目标新变频参数。
* **5. 纱线拉力计算**：输入纤度（D）与强度（g/D），算出纱线能承受的绝对拉力（kg）。
* **6. 纱线需油量计算**：根据第一冷辊速度、丹尼和目标含油量，计算上油量（g/min）。
* **7. 纱线物性与对比**：支持双列并排输入，快速计算两组纱线的 I 值并自动比较大小。
* **8. 挤出与纤度预测 (机台联动)**：联动 1~15 号机台配置，输入转速、速度、孔数，实时预测总挤出量、总丹尼和单纤 DPF。
* **9. 频闪仪线速度换算**：固定辊筒直径后，支持转速（RPM）与线速度（m/min）的双向智能换算。

### 🚀 快速运行
1. 安装依赖：`pip install customtkinter`
2. 运行源码：`python 工业纺丝外汇.py`

## 📅 2. 个人效率工作站

这是我的第二个作品。由于平时折腾的事情越来越多，网页和代办事项切换很麻烦，于是我用代码给自己做了一个定制版的一站式个人效率工作站（Workspace）。

### 🌟 核心功能亮点
* **待办清单管理 (To-Do List)**：左上角常驻待办卡片，支持随时添加、查看并清理日常繁杂的任务。
* **日历排程可视化 (Calendar)**：主体全尺寸月度日历，输入事项名称与日期即可一键上墙显示，支持月份切换。
* **数据安全防丢**：右上角集成“备份数据”、“恢复数据”以及“一键清空”，硬件级排程管理，数据不丢失。
* **高频网页 Dock 栏**：底部快速直达高频 AI 助手（Gemini、ChatGPT、豆包）以及视频平台（Bilibili、抖音）。

## 🔍 3. 本地极速文件搜索器

这是我的第三个作品。Windows 自带的文件资源管理器搜索速度实在让人抓狂，于是我用 Python 写了这个沉浸式的大圆角**全盘/指定目录极速搜索器**。

### 🌟 核心功能亮点
* **强力超清缩放**：强制开启 Windows 底层高 DPI 缩放（`SetProcessDpiAwareness`），彻底告别传统多功能软件在电脑上的模糊与像素低问题。
* **高速全盘索引**：利用 `os.walk` 机制，首次选择文件夹时一键全量收录。测试中几万个文件可在几秒内高速索引完毕，并自动生成本地 JSON 记忆配置文件。
* **沉浸式实时高感度搜索**：搜索框内置 `StringVar` 变量监听，**你每在键盘上敲下一个字母，下方的备选列表就会瞬间过滤刷新**，完全不需要按回车或点搜索按钮。
* **双击无缝定位**：在搜索出的结果列表中，**双击任意行，系统会自动弹出 Windows 自带的资源管理器并直接高亮选中该文件**，找文件、定位一气呵成！

### 🚀 快速运行
1. 安装依赖：`pip install customtkinter`
2. 运行源码：`python 你的搜索器文件名.py`
