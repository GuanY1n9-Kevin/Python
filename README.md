# 纺丝工艺参数计算器 v2.9 (Fiber Spinning Process Calculator)

这是一个专门为**纺织化纤、熔融纺丝（PP/锦纶/涤纶等）车间及研发实验室**量身定制的轻量级桌面工艺计算软件。
软件基于 Python 的 `CustomTkinter` 现代化 GUI 框架开发，界面适配系统深色/浅色模式，排版逻辑完全贴合一线工艺工程师与机台操作工的日常调机和测样习惯。

## 🌟 项目亮点

* **现代美观的 UI**：告别传统 Tkinter 的简陋界面，采用现代扁平化卡片式设计，支持系统级深色模式切换。
* **极简智能换算**：多个模块支持“输入 1 个，留空其余”的智能联动补全，彻底告别繁琐的公式死算。
* **机台硬件数据绑定**：8 号功能区内置工厂 1~15 号常见机台的计量泵孔数和 ml 规格，下拉即可一键调取。
* **实时动态计算**：部分核心模块（如挤出预测）采用变量监听机制，数字随输随显，无需频繁点击按钮。
* **工业级异常防呆**：全软件输入框标配 `try-except` 异常拦截，对留空、误输入字母、分母为零等情况做完美防崩溃处理，并标配“一键清空（保留核心默认值）”功能。

---

## 🛠️ 9 大核心功能模块详解

### 1. 纤度计算 (圈数)
* **用途**：车间摇黑板、缕纱机称重后快速换算。
* **逻辑**：通过测试圈数和天平克重，一键输出精密至小数点后两位的旦尼尔数（D）和分特克斯（dtex）。

### 2. 纺丝产能预估
* **用途**：排产、算大货发货产量。
* **逻辑**：输入卷绕速度、纱线丹尼、锭数以及工作时间，清晰展示推导过程，精准预估整机小时产能（kg/h）及总产量（kg）。

### 3. 长重双向转换
* **用途**：库房发货、车间打样长重对调。
* **逻辑**：支持 D 和 dtex 双单位选择。规格必填后，在“长度/千克/克”中任意填写一项，其余两项全自动联动补全。

### 4. 变频器比例换算
* **用途**：车间改纺、微调规格时调整计量泵频率。
* **逻辑**：基于经典的比例控制法则 $\frac{D_1}{g_1} = \frac{D_2}{g_2}$，输入当前丹尼、当前变频、目标丹尼，秒出目标变频器应调数值。

### 5. 纱线拉力计算
* **用途**：品检物性数据转换。
* **逻辑**：输入纤度（Denier）与强力（g/D），快速算出该纱线可承受的绝对拉力（kg），方便与外贸客户直接对接。

### 6. 纱线需油量计算
* **用途**：纺丝上油（Spin Finish）油剂泵流量设定。
* **逻辑**：根据第一冷辊速度和丹尼数算出每分钟实际出丝克重，再乘以目标含油量（%），精准输出每分钟需要的纯油剂重量（g/min）。

### 7. 纱线物性与对比
* **用途**：新旧批次或不同工艺配方的纱线品质（韧性指数）对比。
* **逻辑**：支持双列并排输入。公式为 $I = \sqrt{\text{深度}\%} \times \text{断裂强度}$。双列填满时自动输出两组 $I$ 值并进行大小比对；次列留空时可作为单项计算器。

### 8. 挤出与纤度预测 (机台联动)
* **用途**：最核心的调机预测模块。
* **逻辑**：下拉选择机台号（1~15号机），系统自动调出对应的泵孔数与 ml/孔 规格。输入转速、熔体密度、卷绕速度及喷丝板孔数，**全自动实时同步**算出：总挤出量 (g/min) $\rightarrow$ 纱线总丹尼 (D) $\rightarrow$ 单纤丹尼 (DPF) 和可产出股数。

### 9. 频闪仪线速度换算
* **用途**：利用频闪仪测定辊筒（冷辊、牵伸辊）实际线速度。
* **逻辑**：双向联动。固定辊筒直径后，输入频闪仪转速（RPM）可秒出线速度（m/min）；反之输入目标线速度也可逆推频闪仪目标闪光频率。
